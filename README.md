# Data Science Labs

Collection of Data Science laboratory exercises completed using Python, Jupyter Notebook, NumPy, Pandas, Matplotlib, Seaborn, SciPy, and Scikit-Learn.

## Topics Covered

- Data Visualization
- Central Limit Theorem
- Hypothesis Testing
- Linear Regression
- Exploratory Data Analysis (EDA)

## Tools Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
